#  ශ්‍රී ලංකා පරිගණක හදිසි ප්‍රතිචාර ඒකකය   (SLCERT)

![](../../../img/logo-SLCERT.jpg)

ශ්‍රී ලංකා CERT | 2006 වර්ෂයේදී ශ්‍රී ලංකා තොරතුරු හා සන්නිවේදන තාක්ෂණ නියෝජිතායතනය (ICTA) විසින් මෙම ඒකකය ස්ථාපිත කරන ලදී. මෙය පිහිටුවීමට ප්‍රධානතම හේතුව වූයේ ශ්‍රී ලංකාවේ තොරතුරු තාක්ෂණ යටිතල පහසුකම් සීඝ්‍රයෙන් වර්ධනය වීම හේතුවෙන් සයිබර් ආරක්ෂණ සිදුවීම්වල ඇති විය හැකි වැඩිවීමට පිළියම් යෙදීමයි. එය පුද්ගලික සීමිත වගකීම් සමාගමක් ලෙස ලියාපදිංචි වී ඇති අතර ශ්‍රී ලංකා CERT|CC දැනට තාක්ෂණ අමාත්‍යාංශය යටතේ සිය ක්‍රියාවලිය සිදු කරයි.

වෙබ් අඩවිය: https://www.cert.gov.lk
